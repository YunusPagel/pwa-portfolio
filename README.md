# PWA Portfolio

A small Progressive Web App built with HTML, CSS, and JavaScript.

## Features
- Responsive user interface
- Simple interactive functionality
- Can be installed as a Progressive Web App on supported devices
- Demonstrates basic web development concepts

## Purpose
This project demonstrates practical web development skills and an understanding of Progressive Web Apps by implementing a simple PWA structure.

## Usage
Open `index.html` in a browser or serve with a local server (e.g., VS Code Live Server extension).

## Technologies
- HTML
- CSS
- JavaScript
